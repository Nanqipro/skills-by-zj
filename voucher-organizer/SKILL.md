# 单据抽凭整理器 (voucher-organizer)

把一批"按**凭证编号**命名的文件夹"（每个里面装着 ZIP 附件 + 散落的封面/凭证 PDF），
转换成一批"按**台账目标列**（如金蝶凭证号）命名的**干净文件夹**"，每个里面只剩从 ZIP 解压出来的单据文件。

这是一个**破坏性**（会删除、重命名文件）但**可预检**的流程。核心原则：**先看清楚，再动手**。

## 何时使用

当用户的任务符合"读取一张 Excel 映射表 → 批量解压文件夹内的 ZIP → 删掉多余文件 → 按映射把文件夹改名"时使用。
典型信号：单据抽凭、凭证/附件归档、台账、金蝶凭证号、银行回单、按编号批量重命名、批量解压。

## 输入与产物

- **台账 Excel**：含"组成文件夹名的若干列"（如 会计年度+凭证类型+凭证编号）和"目标名列"（如 金蝶凭证号）。
- **根目录 root**：里面是一批待处理子文件夹，文件夹名 = 那几列的值用 `-` 拼接（如 `2025-F9-8990000119`）。
- **产物**：每个子文件夹被改名为目标列的值（如 `2025-08-92`），内部只保留 ZIP 解压出的文件；外加一份审计 CSV。

## 工作流（严格按序）

先与用户确认这几项，再开工：台账路径、根目录、组成文件夹名的列及顺序、目标列、是否需要按某列过滤（如公司代码）。
`scripts/config.example.json` 是一份可直接改的配置样例。

### 第 1 步：预检（preflight）—— 绝不能跳过

```bash
python scripts/organize_vouchers.py preflight --config config.json
```

它只读不写，输出待处理数量、多ZIP/无ZIP分布，以及一组**风险检查**（应全为"无"）：
台账缺映射、目标列为空、目标名非法字符、目标名彼此冲突、目标名已存在、映射键冲突。
**把这份报告给用户看。** 出现阻断性风险（重名/冲突/非法字符）时，先让用户核对台账，不要 process。

为什么重要：真实台账常出现"同一凭证编号多行、部分目标为空"，或两个文件夹映射到同一目标名。
盲目执行会覆盖或误删文件。脚本已做了"空值不覆盖真实值""同名不同内容各留一份"等保护，但人工过目仍是最后防线。

### 第 2 步：执行（process）

```bash
python scripts/organize_vouchers.py process --config config.json
```

对每个文件夹：递归解压全部 ZIP（含 ZIP 套 ZIP，如"回单.zip"），删除原有的散落非ZIP文件与ZIP本体，
再把文件夹重命名为目标名。

关键行为（都已内置，不要绕过）：
- **中文文件名**：按 ZIP 的 UTF-8 标志位解码，必要时回退 GBK，避免出现 `щЫ╖` 之类乱码。
- **无ZIP的文件夹**：只删 `.DS_Store`，**保留原文件**后再改名——绝不把文件夹清空（否则丢单据）。
- **嵌套ZIP**：循环解压直到没有 ZIP 为止。
- **同名冲突**：内容相同则去重，内容不同则保留两份（加 `__1` 后缀），不静默覆盖。
- **幂等**：改名后的文件夹不再匹配源正则，可安全重跑。

### 第 3 步：报告（report）

```bash
python scripts/organize_vouchers.py report --config config.json --out 处理结果对照表.csv
```

生成"原文件夹 → 新名 → 文件数 → 文件清单 → 备注"的审计表（UTF-8 BOM，Excel 直接打开）。
做完后**核对**：无残留 `.zip`、无源命名（`原模式`）文件夹遗留、无空文件夹。

## 环境注意

- 依赖 `openpyxl`（`pip install openpyxl --break-system-packages`）；解压用标准库 `zipfile`，无需 `unzip`（`unzip` 在部分环境会显示中文乱码）。
- **删除可能需要授权**：在 Cowork 中若 `os.remove`/`rm` 报 "Operation not permitted"，先调用 `allow_cowork_file_delete` 申请该目录的删除权限，再重跑 process。
- 处理大批量前，建议先在 1–2 个文件夹上小范围验证产物结构符合预期。

## 参数速查

| 配置项 | 说明 |
|--------|------|
| `root` | 待处理子文件夹所在目录（必填） |
| `xlsx` | 台账 Excel 路径（必填） |
| `sheet` | 工作表名；省略则取活动表 |
| `key_cols` | 组成文件夹名的列，顺序敏感，如 `["会计年度","凭证类型","凭证编号"]` |
| `target_col` | 作为新文件夹名的列，如 `金蝶凭证号` |
| `filters` | 行过滤，如 `{"公司代码":"0111"}` |
| `folder_regex` | 识别源文件夹的正则，默认 `^\d{4}-[A-Za-z0-9]+-\d+$` |

命令行同名参数（`--root`/`--xlsx`/`--sheet`/`--key-cols`/`--target-col`/`--filter k=v`/`--folder-regex`）可覆盖 config。
